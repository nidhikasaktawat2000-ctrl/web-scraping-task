# 🌐 Web Scraping Project

This project is part of my Data Analytics Internship at Codalfa.

## 🔹 Project Overview

In this project, I used Python to extract quotes data from a public website and stored it in a CSV file for further analysis.

## 🛠️ Technologies Used

* Python
* Requests
* BeautifulSoup
* CSV

## ⚙️ How It Works

1. Sent a request to the website using the Requests library.
2. Parsed the HTML content using BeautifulSoup.
3. Extracted quotes from the webpage.
4. Stored the extracted data into a CSV file (`quotes.csv`).

## 📂 Output

* A CSV file containing collected quotes.

## 🚀 Key Learning

* Understanding HTML structure
* Data extraction from websites
* Handling real-world data
* Working with Python libraries
